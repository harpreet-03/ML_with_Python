
---> Range functions returns a sequence of numbers, starting from O by default. and increments by
1 (by default). and stops before a specified number.


# range( start?, stop, step?)

eg : for el in range(5)
print(el)

for el in range(1,5)
print(el)

for el in range(1,5,2)
print(el)


---> pass statement

pass is a null statement that does nothing, It is used as a placeholder for future code.

for el in range(10):
    pass
